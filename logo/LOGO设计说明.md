# 城诺科技 LOGO 设计方案

## 一、设计方向概览

提供 **3 个方向** 的 LOGO 设计概念，每个方向都有独特的图形语言，但都围绕「城诺」的核心理念。

---

## 方向一：城池守护

### 图形概念
以**方形/盾牌**为外形轮廓，内部融入：
- "城"字的抽象笔画
- 城堡城门或城垛的简化线条
- 一条上升曲线（代表财富增长）

### 视觉呈现
```
外框：圆角方形（暗示盾牌/城池）
内部：负空间形成的"城"字
底部：渐变绿色上升弧线
```

### 色彩应用
| 元素 | 色彩 |
|------|------|
| 外框 | `#4A90E2` 科技蓝 |
| 负空间 | 透明（背景色） |
| 上升弧线 | `#00C896` → `#00E5B0` 渐变 |

### 适用场景
- 正式商务场合
- 深色背景
- 打印物料

---

## 方向二：环融增长

### 图形概念
以**圆环**为主体，融合：
- 两个相互连接的圆环（合作/社区）
- 环内嵌入箭头（增长/交易）
- 圆形代表全球视野和闭环生态

### 视觉呈现
```
主体：双圆环，线条粗细2px
交叉点：形成箭头向上
配色：蓝绿双色区分两个环
```

### 色彩应用
| 元素 | 色彩 |
|------|------|
| 左环/主环 | `#4A90E2` 科技蓝 |
| 右环/辅环 | `#00C896` 财富绿 |
| 箭头 | `#F5A623` 金色点缀 |

### 适用场景
- 社区/平台属性展示
- 头像/小尺寸使用
- 动态动画效果

---

## 方向三：科技芯片

### 图形概念
以**芯片/电路板**为灵感：
- 方形芯片轮廓
- 内部电路线条汇聚成"城"字
- 角落有数据节点圆点

### 视觉呈现
```
外形：倒圆角方形（芯片封装外形）
内部：线条从四角向中心汇聚
中心：形成"CN"或"城"字的负空间
节点：四角各有1-2个装饰圆点
```

### 色彩应用
| 元素 | 色彩 |
|------|------|
| 外框/线路 | `#4A90E2` 科技蓝 |
| 负空间 | 透明 |
| 节点圆点 | `#00C896` 财富绿 |

### 适用场景
- 技术属性强调
- 网站/APP图标
- 科技感宣传

## 文字LOGO

### 标准文字

```
字体：思源黑体 Bold / 方正兰亭黑 Bold
字重：BOLD (700)
字间距：收紧 5%
颜色：#E6EDF3 (浅色背景) / #4A90E2 (深色背景)
```

### 标语文字

```
字体：思源黑体 Regular / Medium
字重：400-500
字间距：正常
颜色：#8B949E
```

### 组合方式

| 类型 | 说明 |
|------|------|
| 横向 | LOGO图标 + 文字 + 标语（左到右） |
| 纵向 | LOGO图标 + 文字 + 标语（上到下） |
| 纯图标 | 仅图形，用于头像等小尺寸 |

## 使用示例

### 深色背景（推荐）
```
背景色：#0d1117 或 #161b22
LOGO色：白色 #E6EDF3 或 渐变蓝绿
效果：高档科技感
```

### 浅色背景
```
背景色：#FFFFFF 或 #F6F8FA
LOGO色：深蓝 #4A90E2 + 深绿 #00C896
效果：干净专业
```

### 按钮/CTA场景
```
背景：蓝绿渐变
LOGO：全白色
效果：行动号召
```
