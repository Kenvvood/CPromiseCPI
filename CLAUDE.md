# 城诺科技 网站项目开发准则

## 文件管理规范

### 核心文件
- **`手册/招商手册.html`** - 完整招商手册，用于 Netlify 部署
- **`h5/index.html`** - H5 移动端版本
- **`dist/index.html`** - Netlify 部署文件夹（基于招商手册生成）

### 同步要求
**所有代码更新必须同时修改以下两个文件，保持完全同步：**
1. `手册/招商手册.html`
2. `h5/index.html`

部署前，将 `手册/招商手册.html` 复制为 `dist/index.html` 后上传至 Netlify。

## 页面结构规范

### 导航栏
| 设备 | 顶部导航 |
|------|---------|
| 桌面 (≥768px) | 关于我们 \| 核心价值 \| 合作项目 \| **立即咨询**（金色按钮） |
| 手机 (<768px) | Logo + **立即咨询**（金色按钮） |

- 移动端**不显示**底部导航栏
- 移动端顶部**必须保留**"立即咨询"金色按钮

位于"策略介绍"和"核心优势"之间，三行居中排列：

```
    ─────────────────────
     专业稳健长期主义
    城中立诺，信满八方  ← 渐变色突出显示
     城诺科技，专注量化投资与智能交易
     用技术创造价值，让信任驱动增长
    ─────────────────────
```

## 响应式设计原则

- **单链接自适应**：一个 URL 适配所有设备，无需分链接
- 使用 CSS 媒体查询 `@media (max-width: 768px)` 和 `@media (max-width: 480px)`
- 移动端优先考虑触摸交互和大按钮

## Netlify 部署指南

### 部署结构
```
dist/
└── index.html    ← 直接上传整个文件夹
```

### 部署步骤
1. 将 `手册/招商手册.html` 复制为 `dist/index.html`
2. 登录 [app.netlify.com](https://app.netlify.com)
3. 点击 **Add new site** → **Deploy manually**
4. 拖拽 `dist` 文件夹上传
5. 等待部署完成
