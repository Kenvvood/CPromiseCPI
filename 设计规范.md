## 二、色彩系统

### 主色板

| 名称 | 色值 | 用途 |
|------|------|------|
| **深空蓝** | `#0d1117` | 主背景色 |
| **暗夜蓝** | `#161b22` | 卡片/区块背景 |
| **科技蓝** | `#4A90E2` | 主强调色、按钮、链接 |
| **财富绿** | `#00C896` | 增长、正收益、成功状态 |
| **金色点缀** | `#F5A623` | 重要数据、高亮（少量使用） |
| **文字白** | `#E6EDF3` | 主标题文字 |
| **文字灰** | `#8B949E` | 次要文字、说明文字 |
| **边框灰** | `#30363D` | 分隔线、边框 |

### 渐变规范

```
背景渐变：linear-gradient(135deg, #0d1117 0%, #161b22 100%)
卡片光效：linear-gradient(180deg, rgba(74, 144, 226, 0.1) 0%, transparent 100%)
按钮渐变：linear-gradient(90deg, #4A90E2 0%, #00C896 100%)
```

### 语义色彩

| 场景 | 色值 |
|------|------|

| 风险/警示 | `#F85149` |
| 数据图表-蓝 | `#4A90E2` |
| 数据图表-绿 | `#00C896` |

## 三、字体规范

### 字体家族

| 用途 | 字体 | 备选 |
|------|------|------|
| **标题** | 思源黑体 (Source Han Sans CN) | 阿里巴巴普惠体 |
| **正文** | 思源黑体 / 系统默认无衬线 | - |
| **数字/代码** | JetBrains Mono / Roboto Mono | Consolas |

| 层级 | 字号 | 字重 | 行高 |

## 四、图标规范

### 图标风格

- **类型**：线性图标（Line Icons）
- **描边宽度**：2px
- **圆角**：2px（统一）
- **风格**：参考 GitHub Desktop / Lucide Icons
- **颜色**：默认 `#8B949E`，hover/active 时为 `#4A90E2`

### 常用图标建议

| 场景 | 图标方向 |
|------|----------|
| 安全/盾牌 | 城池、盾牌、锁 |
| 增长/收益 | 上升箭头、折线图 |
| 算法/技术 | 芯片、代码、数据流 |
| 社区/团队 | 人形图标、握手 |
| 全球化 | 地球、环形箭头 |
| 财富 | 金币、钻石、皇冠 |

## 五、卡片与容器

### 卡片规范

```css
.card {
  background: #161b22;
  border: 1px solid #30363D;
  border-radius: 12px;
  padding: 24px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.3);
}

/* 悬停状态 */
.card:hover {
  border-color: #4A90E2;
  box-shadow: 0 0 20px rgba(74, 144, 226, 0.2);
}
```

## 六、数据可视化

### 图表风格

- **背景**：透明或 `#161b22`
- **线条颜色**：使用语义色彩（蓝、绿、金）
- **线条粗细**：2px
- **节点**：圆形，半径 4px
- **文字**：14px，#8B949E

### 建议图表类型

| 数据类型 | 推荐图表 |
|----------|----------|
| 资金曲线 | 平滑折线图，带渐变填充 |
| 收益对比 | 柱状图 |
| 策略分布 | 环形饼图 |
| 业绩指标 | 数字卡片 + 趋势箭头 |

---

## 七、LOGO 使用规范

### 最小尺寸

- 主LOGO最小宽度：80px
- 图标版最小宽度：32px

### 安全空间

- LOGO周围保留 1/4 LOGO宽度的安全空间

### 背景适配

| 浅色背景 | 深色背景 | 彩色背景 |
|----------|----------|----------|
| 使用深色版本 | 使用浅色/白色版本 | 使用白色版本 + 描边 |
